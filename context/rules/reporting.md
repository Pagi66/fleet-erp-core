LOGS AND REPORTING RULES
RULE R-001: Daily Log Enforcement
At the end of each day:
Engine Room Register must be completed
Equipment Operation Record must be updated
RULE R-002: Log Verification
All daily logs must:
Be signed by the MEO
Be available for CO review
RULE R-003: Weekly Reporting Compliance
All required weekly reports shall:
Be automatically compiled
Be submitted to FSG and Log Comd
RULE R-004: Equipment Usage Tracking
Operating hours shall:
Be recorded daily
Be used to trigger maintenance cycles
RULE R-005: Steady Steaming Reporting
If ship operates for 2 hours or more:
Performance report must be generated
Fuel and load data must be recorded
RULE R-006: Technical Reporting
Any deficiency in material, design, or support shall:
Be reported using formal technical report
Be reviewed by FSG and Log Comd
RULE R-007: Maintenance Request Submission
When ship cannot execute maintenance:
Maintenance Request must be submitted
Must include full defect details
Must be tracked until closure
