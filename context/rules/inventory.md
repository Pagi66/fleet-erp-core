INVENTORY AND STORES RULES
RULE I-001: Inventory Sufficiency
Ships must maintain:
Minimum 3 month stock for maintenance
RULE I-002: Stock Monitoring
System shall continuously monitor:
Spare parts levels
Consumption rates
RULE I-003: Auto Requisition
If stock falls below threshold:
Spare parts request must be generated
RULE I-004: Store Accountability
All technical stores shall:
Be recorded
Be auditable
Be available during inspection
